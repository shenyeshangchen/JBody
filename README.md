## 功能介绍 

![输入图片说明](demo/Untitled.png)

健身房小程序：为健身房打造的轻量化服务平台，旨在提升会员健身体验、优化场馆运营效率，实现线上线下服务闭环。它既为会员提供便捷的信息获取、课程与场地预约、个人中心管理等功能，也为健身房管理员提供高效的内容维护、预约管理、用户管理及数据导出等后台能力。对会员：信息获取更及时，课程预约更便捷，健身管理更清晰。对健身房：运营管理更高效，数据统计更精准，服务质量可量化。
 ![输入图片说明](demo/%E5%81%A5%E8%BA%AB%E6%88%BF%E5%B0%8F%E7%A8%8B%E5%BA%8F%E5%B0%8F%E7%A8%8B%E5%BA%8F%E6%A6%82%E8%A6%81%E8%AE%BE%E8%AE%A1.jpg)

## 技术运用
- 前端基于微信小程序平台进行开发
- 后端基于Java Springboot架构开发
- 数据库： MySQL (8.0+)
